认识CSS

定义
层叠样式表 (Cascading Style Sheets)
作用
  • 用于定义HTML内容在浏览器内的显示样式，如文字大小、颜色、字体加粗等
  • 解决内容与表现分离的问题
  • 极大地提高了工作效率

CSS语法

CSS样式规则

CSS样式由选择符和声明组成，而声明又由属性和值组成，如

p {
    color: red;
    font-size : 12px;
}

'p'为选择符

大括号之间的内容为声明

'color','font-size'为属性

'red','12px'为值

CSS代码注释

用/*注释语句*/来标明，如

/*我是注释啦啦啦*/
p {
    color: red;
    font-size : 12px;
}

CSS使用方式

内联式css样式

把CSS代码直接写在现有的HTML标签中，如

<p style="color:red; font-size:12px">这里文字是红色。</p>

嵌入式css样式

把css样式代码写在<style type="text/css"></style>"标签之间，如

<style type="text/css">
span {
    color: red;
}
</style>


一般情况下嵌入式css样式写在<head></head>之间

CSS使用方法

外部式css样式

把css代码写一个单独的外部文件中，这个css样式文件以“.css”为扩展名，在<head>内（不是在<style>标签内）使用<link>标签将css样式文件链接到HTML文件内，如

<link href="base.css" rel="stylesheet" type="text/css" />

注意：<link>标签位置一般写在<head>标签之内

三种方法的优先级

当内联式、嵌入式、外部式样式表中css样式是在的相同权值的情况下且嵌入式css样式的位置在外部式的后面时，有

内联式>嵌入式>外部式（就近原则）

常用基本样式-背景

背景颜色

background-color不能继承（所有背景属性都不能继承），其默认值是transparent

可以为所有元素设置背景色，这包括 body 一直到 em 和 a 等行内元素，如

p {
    background-color: gray;
}

背景图像

图片路径是相对于CSS的位置

p {
    background-image: url(../img/text.png);
}

背景大小(CSS3)

重新设置背景图片大小

p {
    background-image: url(../img/text.png);
    background-size: 60px 50px;
}

常用基本样式-背景

背景重复

在页面上对背景图像进行平铺

p {
    background-image: url(../img/text.png);
    background-repeat: no-repeat;
}


  • 属性值repeat导致图像在水平垂直方向上都平铺(默认)
  • repeat-x 和 repeat-y 分别导致图像只在水平或垂直方向上重复
  • no-repeat 则不允许图像在任何方向上平铺
背景定位

改变图像在背景中的位置

p {
    background-image: url(../img/text.png);
    background-repeat: no-repeat;
    background-position: 10% 20%;
}


  • 可以使用一些关键字：top、bottom、left、right和center
  • 属性值不能超过两个（水平方向和垂直方向）
  • 可以使用长度值，如 100px 或 5cm
  • 可以使用百分数值

常用基本样式-字体和文本

字体系列

p {
    font-family: "Microsoft YaHei", serif;
}

  • 用户代理会按所列的顺序查找这些字体
  • 当字体名中有一个或多个空格，使用引号
字体粗细

p {
    font-weight: normal;
}

  • 关键字 100 ~ 900 为字体指定了 9 级加粗度
  • 数字 400 等价于 normal，而 700 等价于 bold
字体大小

p {
    font-size: 14px;
}

  • 普通文本（比如段落）的默认大小是 16 像素
  • 1em 等于当前的字体尺寸
  • 也能使用百分比
常用基本样式-超链接
设置链接的样式
  • a:link - 普通的、未被访问的链接
  • a:visited - 用户已访问的链接
  • a:hover - 鼠标指针位于链接的上方
  • a:active - 链接被点击的时刻

a:link {
    color: red;
    text-decoration: none;
}

当为链接的不同状态设置样式时，请按照以下次序规则：

  • a:hover 必须位于 a:link 和 a:visited 之后
  • a:active 必须位于 a:hover 之后

常用基本样式-列表

列表类型

在一个无序列表中，列表项的标志是出现在各列表项旁边的圆点。在有序列表中，标志可能是字母、数字或另外某种计数体系中的一个符号

ul {
    list-style-type: square;
}


list-style-type修改用于列表项的标志类型：circle、decimal

列表项图像

使用图像作为列表项的标志

ul li {
    list-style-image: url(../img/test.png);
}

选择器

什么是选择器

"选择器"指明了{}中的"样式"的作用对象，也就是“样式”作用于网页中的哪些元素

每一条css样式声明（定义）由两部分组成，形式如下：

选择器 {
    样式;
}

标签选择器

标签选择器其实就是html代码中的标签，通配符'*'表示所有元素(通用选择器)

p {
    color: red;
    font-size : 12px;
}

选择器

类选择器

语法:  .类选器名称{css样式代码;}

.test {
    color: red;
    font-size : 12px;
}


设置class为'test'的标签，如

    <p class="test">纳尼？我被盯上了</p>

ID选择器

语法:  #类选器名称{css样式代码;}

#test {
    color: red;
    font-size : 12px;
}

设置id为'test'的标签的样式，如

    <p id="test">纳尼？我又被盯上了</p>

标签可以有相同的class，但不能有相同的id

选择器

子选择器

语法:  父选择器 > 子选择器{css样式代码;}，指定父元素的第一代子元素

.test > p {
    color: red;
    font-size : 12px;
}


指定'test'类的标签元素下第一代<p>标签，如

后代选择器

语法:父选择器 子选择器{css样式代码;}，指定父元素的后辈元素

.test div {
    color: red;
    font-size : 12px;
}

指定'test'类的标签元素下所有<p>标签，如

<div class="test">
    <div>纳尼？我被盯上了<div>卧槽？我都被盯上4次了</div></div>
</div>

选择器

相邻兄弟选择器

语法:  A选择器 + B选择器{css样式代码;}，指定紧接在A选择器后的B选择器

h1 + p {
    color: red;
    font-size : 12px;
}

指定紧接在<h1>后的<p>元素，如

    <h1>我是A</h1>

    <p>我是B</p>

伪类选择器

语法:  选择器:伪类{css样式代码;}

p:hover {
    color: red;
    font-size : 12px;
}


当在<p>标签上触发hover事件时,指定相应的样式

选择器

分组选择器

语法:  A选择器, B选择器{css样式代码;}

.test1，.test2 {
    color: red;
    font-size : 12px;
}


相当于A选择器{CSS样式代码;}和B选择器{CSS样式代码;}

多类选择器

语法:  .A类选择器.B类选择器{css样式代码;}，同时指定A,B选择器

.test1.test2 {
    color: red;
    font-size : 12px;
}


指定同时具有.test1类和.test2类的元素，如

    <div class="test1 test2">我有2个类你怕不怕</div>

选择器

特殊性

标签的权值为1，类选择符的权值为10，ID选择符的权值最高为100

p {color:red;} /*权值为1*/

p span {color:green;} /*权值为1+1=2*/

.warning {color:white;} /*权值为10*/


当同一个元素设置了不同的CSS样式代码,选用权值最高的样式

层叠

层叠就是在html文件中对于同一个元素可以有多个css样式存在，当有相同权重的样式存在时，会根据这些css样式的前后顺序来决定，处于最后面的css样式会被应用

p {color: red;}

p {color: black;}


<p>最终字体颜色为black

重要性

使用!important为某些样式设置具有最高权值，如：p {color: red!important;}

盒模型-框模型

框模型
内边距
  • padding 属性定义元素边框与元素内容之间的空白区域
  • 接受长度值或百分比值，但不允许使用负值
边框
  • 围绕元素内容和内边距的一条或多条线
  • border 属性允许你规定元素边框的样式、宽度和颜色
外边距
  • 围绕在元素边框的空白区域是外边距,设置外边距会在元素外创建额外的“空白”
  • 接受长度值或百分比值，可以设置auto和负值

盒模型-box-sizing(CSS3)

content-box（标准W3C盒子模型）

宽度和高度分别应用到元素的内容框，在宽度和高度之外绘制元素的内边距和边框

border-box（IE盒子模型）

为元素设定的宽度和高度决定了元素的边框盒。就是说，为元素指定的任何内边距和边框都将在已设定的宽度和高度内进行绘制，通过从已设定的宽度和高度分别减去边框和内边距才能得到内容的宽度和高度。

div {
    box-sizing: border-box;
    -webkit-box-sizing: border-box;
}

标签元素-简单介绍

元素分类
  • 块状元素
  • 内联元素（又叫行内元素）
  • 内联块状元素
常用的块状元素

<div>、<p>、<h1>...<h6>、<ol>、<ul>、<dl>、<table>、<address>、<blockquote> 、<form>

常用的内联元素

<a>、<span>、<br>、<i>、<em>、<strong>、<label>、<q>、<var>、<cite>、<code>

常用的内联块状元素

<img>、<input>、<a>、<span>

标签元素-简单介绍

块状元素特点
  • 每个块级元素都从新的一行开始，并且其后的元素也另起一行
  • 元素的高度、宽度、行高以及顶和底边距都可设置
  • 元素宽度在不设置的情况下，是它本身父容器的100%（和父元素的宽度一致），除非设定一个宽度
内联元素特点
  • 和其他元素都在一行上
  • 元素的高度、宽度及顶部和底部边距不可设置
  • 元素的宽度就是它包含的文字或图片的宽度，不可改变
内联块状元素特点
  • 和其他元素都在一行上
  • 元素的高度、宽度、行高以及顶和底边距都可设置

CSS布局模型-流动模型(Flow)

流动模型(Flow)
流动（Flow）是默认的网页布局模式
特点
  • 块状元素都会在所处的包含元素内自上而下按顺序垂直延伸分布，因为在默认状态下，块状元素的宽度都为100%
  • 内联元素都会在所处的包含元素内从左到右水平分布显示

CSS布局模型-浮动模型(Float)

浮动模型(Float)
通过float属性设置
特点
  • 浮动的框可以向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止
  • 由于浮动框不在文档的普通流中，所以文档的普通流中的块框表现得就像浮动框不存在一样

如

div {
    width: 100px;
    height: 100px;
    background: white;
    border: 4px solid black;
    float: left;
}

CSS布局模型-层模型(Layer)

层模型(Layer)

每个框能够精确定位操作，通过修改position属性改变层布局

  • static（默认）-元素框正常生成。块级元素生成一个矩形框，作为文档流的一部分，行内元素则会创建一个或多个行框，置于其父元素中
  • relative-元素框偏移某个距离。元素仍保持其未定位前的形状，它原本所占的空间仍保留
  • absolute-元素框从文档流完全删除，并相对于其包含块定位。包含块是最近的position值不为static的祖先元素
  • fixed-元素框的表现类似于将 position 设置为 absolute，不过其包含块是视窗本身

CSS3-转换

2D转换

通过 CSS3 转换(transform属性,Chrome 和 Safari 需要前缀 -webkit-)，我们能够对元素进行移动、缩放、转动、拉长或拉伸

移动-translate()

元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数

转动-rotate()

通过 rotate() 方法，元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转

缩放-scale()

通过 scale() 方法，元素的尺寸会增加或减少，根据给定的宽度（X 轴）和高度（Y 轴）参数

CSS3-过渡

介绍

通过 CSS3（transition 属性， Safari 需要前缀 -webkit-），我们可以在不使用 Flash 动画或 JavaScript 的情况下，当元素从一种样式变换为另一种样式时为元素添加效果

使用
  • 规定您希望把效果添加到哪个 CSS 属性上
  • 规定效果的时长，默认为0
  • 规定过渡效果的时间曲线，默认是 "ease"
  • 规定过渡效果何时开始，默认为0

如

div {
    transition-property: width;
    transition-duration: 1s;
    transition-timing-function: ease;
    transition-delay: 2s;
}

等价于

div {
    transition: width 1s ease 2s;
}

CSS3-动画

介绍

通过 CSS3，我们能够创建动画，这可以在许多网页中取代动画图片、Flash 动画以及 JavaScript

使用
  • 学习 @keyframes 规则-@keyframes 规则用于创建动画。在 @keyframes 中规定某项 CSS 样式，就能创建由当前样式逐渐改为新样式的动画效果
  • 把规则捆绑到某个选择器，否则不会产生动画效果

如

编写动画规则

@keyframes myfirst {
    0% {background: red;}
    25% {background: yellow;}
    50% {background: blue;}
    100% {background: green;}
}

捆绑选择器

div {
    animation: myfirst 5s;
}
